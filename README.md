This is a simple Tic Tac Toe game implemented in Java. It allows two players to play the classic game of Tic Tac Toe on a 3x3 grid.

## Features

Two-player gameplay.

Simple text-based interface.

Automatic check for win or draw conditions.

Player 1 is 'X' and Player 2 is 'O'.


## Requirements

Java version: JDK 8 or higher.


## How to Run

1. Clone the repository:

git clone https://github.com/yourusername/tic-tac-toe-java.git


2. Navigate to the project folder:

cd tic-tac-toe-java


3. Compile and run the program:

javac TicTacToe.java
java TicTacToe



## Usage

The game prompts each player to input their move by choosing an empty cell.

The game checks for a winner after each move.

The game ends when there is a winner or a draw.
